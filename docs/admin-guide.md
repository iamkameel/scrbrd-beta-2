# Admin User Guide

## Welcome, Administrator!
This guide will help you manage the SCRBRD cricket management system.

---

## 1. User Management

### Inviting New Users
1. Navigate to **User Management** from the sidebar
2. Click **"Invite User"** button
3. Fill in the form:
   - Email address
   - First name and last name
   - Select role from dropdown
   - Assign schools (if applicable)
4. Click **"Send Invitation"**
5. User will receive an email to set up their account

### Available Roles
- **Admin**: Full system access
- **Sportsmaster**: Manage matches, teams, players
- **Head Coach**: Manage own school's teams
- **Assistant Coach**: View and assist with coaching
- **Scorer**: Record match scores
- **Player**: View own stats and team info
- **Parent**: View child's information

### Editing User Roles
1. Go to **User Management**
2. Find the user in the table
3. Click the **Edit** icon
4. Update role or assigned schools
5. Save changes

---

## 2. School Setup

### Adding a New School
1. Navigate to **Schools** → **Add School**
2. Enter required information:
   - School name
   - Abbreviation (e.g., "WBHS")
   - Province and division
   - Contact details
3. Upload school logo/crest
4. Set brand colors (primary/secondary)
5. Click **"Create School"**

### Managing School Details
- Edit contact information
- Update logo or colors
- Assign teams and players
- View school statistics

---

## 3. Team Configuration

### Creating Teams
1. Go to **Teams** → **Add Team**
2. Select parent school
3. Enter team details:
   - Team name
   - Suffix (1st XI, U15A, etc.)
   - Division/age group
4. Assign captain and coach
5. Save team

### Managing Team Rosters
1. Open team details page
2. Click **"Manage Roster"**
3. Add players from the school
4. Set batting order
5. Assign jersey numbers

---

## 4. League & Season Management

### Creating a League
1. Navigate to **Leagues** → **Add League**
2. Enter league name
3. Select province
4. Choose type (League/Series/Cup/Friendly)
5. Create league

### Setting Up a Season
1. Go to **Seasons** → **Add Season**
2. Enter season name (e.g., "2025")
3. Set start and end dates
4. Mark as active
5. Save season

### Creating Divisions
1. Within a league, click **"Add Division"**
2. Enter division name
3. Select age group
4. Link to season
5. Add teams to division

---

## 5. Match Scheduling

### Creating a Match
1. Navigate to **Matches** → **Add Match**
2. Fill in match details:
   - Home and away teams
   - Date and time
   - Venue (field)
   - Match type (T20/ODI)
   - Number of overs
3. Assign officials (umpires, scorer)
4. Save match

### Match States
- **SCHEDULED**: Match created, awaiting team selection
- **TEAM_SELECTION**: Teams being finalized
- **PRE_MATCH**: Toss and pre-match checks
- **LIVE**: Match in progress
- **COMPLETED**: Match finished

---

## 6. Field Management

### Adding a Venue
1. Go to **Fields** → **Add Field**
2. Enter field details:
   - Name and location
   - Owning school (optional)
   - GPS coordinates
3. Add facilities:
   - Capacity
   - Floodlights
   - Scoreboard type
4. Set pitch type and condition
5. Save field

---

- Adjust as needed for your organization

### Data Management
- Export data for backup

## Tips for Success

✅ **Keep user roles up to date** - Remove access for users who leave
✅ **Verify match details** - Double-check dates, times, and venues
✅ **Maintain school data** - Keep contact info and logos current
✅ **Regular backups** - Export important data periodically
✅ **Train scorers** - Ensure scorers know how to use the live scoring interface
