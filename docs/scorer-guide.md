# Scorer User Guide

## Welcome, Scorer!
This guide will help you record live cricket matches using the SCRBRD scoring interface.

---

## 1. Pre-Match Workflow

### Accessing the Match
1. Navigate to **Matches** from the sidebar
2. Find your assigned match
3. Click on the match to open details
4. Click **"Pre-Match"** button

### Pre-Match Checklist
Complete these tasks before starting:
- âœ… Verify team lineups (11 players each)
- âœ… Confirm captain and vice-captain
- âœ… Record the toss (winner and decision)
- âœ… Set batting order for first innings
- âœ… Check scorebook is prepared
- âœ… Ensure equipment is ready

### Conducting the Toss
1. In Pre-Match page, find **"Toss"** section
2. Select which team won the toss
3. Choose their decision (Bat or Field)
4. Click **"Record Toss"**

### Setting Batting Order
1. Go to **"Batting Order"** section
2. Drag and drop players into batting positions
3. Ensure opening batsmen are in positions 1 and 2
4. Click **"Save Batting Order"**

---

## 2. Starting the Match

### Launching Live Scoring
1. From match details, click **"Start Scoring"**
2. You'll be taken to the **Live Scoring Interface**
3. The scoreboard will display at the top

### Selecting Players
When the match starts, you need to select:
- **Striker**: The batsman on strike
- **Non-Striker**: The batsman at the other end
- **Bowler**: The current bowler

**How to Select**:
1. Click **"Players"** button
2. Choose striker from batting team
3. Choose non-striker from batting team
4. Choose bowler from bowling team
5. Click **"Confirm"**

---

## 3. Recording Balls

### Using the Wagon Wheel
The **Wagon Wheel** on the right shows where shots are played.

**To record a ball with shot location**:
1. Tap on the wagon wheel where the ball went
2. The scoring dialog will open with coordinates pre-filled

### Using Quick Actions
For faster scoring without shot location:
1. Click **"Record Ball"** button
2. Scoring dialog opens

### Scoring Dialog Options

**Runs**:
- Select runs scored: 0, 1, 2, 3, 4, 6
- Dot ball = 0 runs

**Extras**:
- Wide
- No-ball
- Bye
- Leg-bye
- Penalty runs

**Wicket**:
- Check "Wicket" if batsman is out
- Select dismissal type:
  - Bowled
  - Caught
  - LBW
  - Run Out
  - Stumped
  - Hit Wicket
- Select fielder (if applicable)

**Ball Details** (Optional):
- Length: Full, Good, Short, Yorker
- Line: Off Stump, Middle, Leg, Wide Outside Off

**Submit**:
- Click **"Record Ball"** to save

---

## 4. Managing the Match

### After a Wicket
1. System will prompt you to select new batsman
2. Click **"Players"** button
3. Choose the next batsman from the batting order
4. Click **"Confirm"**

### End of Over
1. When 6 legal balls are bowled, the over ends
2. System will prompt for new bowler
3. Click **"Players"** button
4. Select the new bowler
5. Batsmen swap ends automatically

### Swapping Ends
If batsmen need to swap (e.g., after odd runs):
1. Click **"Swap Ends"** button
2. Striker and non-striker switch

### Undo Last Ball
Made a mistake?
1. Click **"Undo"** button
2. Review the last ball details
3. Click **"Confirm Undo"**
4. The ball will be removed

---

## 5. Ending an Innings

### When to End Innings
- All 10 wickets have fallen (all out)
- Maximum overs completed
- Team declares (rare)

### How to End Innings
1. Click **"End Innings"** button
2. Confirm the action
3. System calculates:
   - Final score
   - Target (if first innings)
   - Match status

### Innings Break
- Take a break
- Prepare for second innings
- Set batting order for chasing team
- Select opening batsmen and bowler

---

## 6. Second Innings

### Starting Second Innings
1. After innings break, select players:
   - New striker (chasing team)
   - New non-striker (chasing team)
   - New bowler (defending team)
2. Begin recording balls as before

### Target Display
- Scoreboard shows **TARGET** prominently
- Runs required updates automatically
- Required run rate is calculated

---

## 7. Completing the Match

### Match Completion
The match ends when:
- Chasing team reaches target (win)
- Chasing team is all out (loss)
- Overs are completed (win/loss/tie)

### Finalizing Results
1. Click **"End Innings"** after second innings
2. System determines winner
3. Match result is displayed
4. Review final scorecard

## 8. Scoreboard Features

### Live Scoreboard Display
The scoreboard shows:
- **Team Names**: Home vs Away
- **Score**: Runs/Wickets (Overs)
- **Target**: For second innings
- **Batsmen**: Current batsmen with runs and balls
- **Bowler**: Current bowler with figures
- **This Over**: Last 6 balls with color coding

## 10. Common Scenarios

### Wide Ball
1. Select "Wide" in extras
2. Add any runs scored off the wide
3. Record ball (does not count toward over)

### No-Ball
1. Select "No-Ball" in extras
2. Add runs scored
3. Check if wicket (only run-out possible)
4. Record ball (does not count toward over)

### Run Out
1. Check "Wicket"
2. Select "Run Out"
3. Choose which batsman is out
4. Select fielder who threw the ball
5. Record ball

### Boundary
1. Select 4 or 6 runs
2. Tap wagon wheel location (optional)
3. Record ball
